Utiliser le nom Mindwtr.

Mindwtr est gratuit et open source, et les forks sont vraiment les bienvenus. La seule chose que la licence du code ne couvre pas, c’est le nom et le logo — voici, en termes simples, ce que cela signifie.

Le code est libre. Le nom n’en fait pas partie.

Le code source de Mindwtr est distribué sous licence AGPL-3.0 — vous pouvez l’utiliser, l’étudier, le modifier et le partager selon ces conditions. C’est tout le principe de l’open source, et nous sommes ravis de vous accueillir. En revanche, l’AGPL ne vous accorde pas le droit d’utiliser le nom ou le logo Mindwtr : les licences logicielles couvrent le code, pas les marques, et le nom comme l’icône sont protégés séparément au titre du droit des marques. C’est normal pour les projets open source, et cela permet à “Mindwtr” de continuer à désigner cette application.

Créez un fork — vraiment.

Créez un fork, corrigez-le, développez votre propre version, utilisez-la pour vous-même ou votre équipe — tout cela est encouragé. Si vous la gardez pour vous ou renvoyez vos modifications au projet d’origine, vous n’avez rien d’autre à prévoir.

Vous la distribuez ? Donnez-lui votre propre nom.

Si vous distribuez une version modifiée à d’autres personnes — votre propre version, une fiche sur une boutique d’applications, une version hébergée — donnez-lui votre propre nom et votre propre icône, et ne la présentez pas sous le nom “Mindwtr” ni comme une version approuvée par ce projet ou affiliée à celui-ci. Un autre nom évite toute confusion pour les utilisateurs et joue franc jeu envers les deux projets.

Un doute ? Demandez-nous.

Vous souhaitez utiliser le nom ou le logo dans un article, une vidéo, un outil compatible ou un projet communautaire ? La plupart des usages raisonnables qui ne prêtent pas à confusion ne posent aucun problème, et nous vous donnerons volontiers notre accord. Écrivez à support@mindwtr.app pour nous poser la question.
