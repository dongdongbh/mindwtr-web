# Mobile installation

Mindwtr is available on iPhone, iPad, and Android. Mobile builds keep the same local-first model as desktop: your data starts on the device, and sync is optional.

## iPhone and iPad

Install Mindwtr from the App Store. After first launch, review notification permissions and sync settings before relying on reminders or cross-device data.

## Android

Choose the channel that fits your device:

| Channel | Use it when |
| --- | --- |
| Google Play | You want the most familiar Android install and update flow. |
| F-Droid | You prefer the main F-Droid repository. |
| IzzyOnDroid | You want an F-Droid-compatible repository with faster availability for some builds. |
| APK from GitHub Releases | You need a direct artifact or are testing a specific release. |

When sideloading an APK, Android may ask you to allow installs from the app that opened the file. Only install APKs from the official GitHub Releases page.

## Data location

Mindwtr stores app data locally in the platform's private app storage. Removing the app can remove local data, depending on platform and uninstall settings. Export a backup before uninstalling, reinstalling, or testing sync changes.

## Capture from other apps

Mobile capture can come from normal in-app entry, share sheets, widgets, shortcuts, URL schemes, and automation surfaces depending on platform support.
