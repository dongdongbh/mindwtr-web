# Desktop guide

Mindwtr desktop is built for focused daily planning, fast capture, and full access to your GTD system.

## Main views

| View | Use it for |
| --- | --- |
| Focus | Today, next actions, and calendar-aware work |
| Inbox | Unprocessed captures |
| Projects | Outcomes that need multiple actions |
| Next Actions | Actions ready to do |
| Waiting For | Delegated or blocked work |
| Someday | Deferred ideas and maybes |
| Review | Weekly review and system cleanup |

## Capture

Use the capture field for quick entry. Plain text is enough. Add quick syntax when it saves time:

- `@context` for contexts
- `#tag` for tags
- `+Project` for project assignment
- `/due:friday` for due dates
- `/next` for next actions
- `/note:details` for notes

## Clarify and organize

Open a task to add notes, dates, context, project, tags, attachments, or review-related metadata. Mindwtr keeps capture lightweight and puts the deeper decisions in task editing and review.

## Focus

Focus is the daily surface. It is meant to answer one question: what can you act on now?

Use Focus for:

- Today and overdue items
- Calendar-linked work
- Starred or selected next actions
- Items that need attention before the next review

## Review

Review helps you keep the system trustworthy. Move through inbox processing, waiting items, projects, and next actions. The review flow is where vague captures become clear decisions.

## Sync and backup

Before changing sync settings, export a backup. For sync setup details, read [Data and sync](/data-sync/).
