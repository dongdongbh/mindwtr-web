# 重复任务

对于完成后应再次出现的工作，请使用重复任务，例如回顾、家务、日常例行事项、续期和维护任务。

重复规则位于任务编辑器中。如果使用快速记录，请先创建任务，再打开编辑。

## 设置重复任务

1. 打开任务。
2. 编辑任务。
3. 在时间安排部分找到**重复**字段。
4. 选择**每天**、**每周**、**每月**或**每年**。
5. 调整重复细节。
6. 保存任务。

桌面端和移动端的任务编辑界面都会显示重复控件。

## 重复选项

| 选项 | 适用事项 |
| --- | --- |
| **每天** | 习惯与每日例行事项。 |
| **每周** | 每周回顾、会议、杂事或维护。 |
| **每月** | 账单、月度回顾、订阅或月度检查清单。 |
| **每年** | 年度续期、税务、登记和纪念日。 |

每种重复类型都可通过**每隔**跳过一定间隔，例如每 2 周或每 3 个月。

每周任务可选择具体工作日。每月任务可在每月同一天重复，也可使用**自定义**设置特定星期几等模式。

## 固定日程还是完成后重复

默认情况下，Mindwtr 保持固定日程。适用于应始终与日历日期绑定的截止事项或承诺。

示例：

- 每月交房租。
- 每月第一个星期六检查财务。
- 每年续保险。

如果下一项任务应根据当前任务实际完成的时间安排，请开启**完成后重复**。

示例：

- 浇水后 3 天再次浇花。
- 完成清洁后 2 周再次清洁过滤器。
- 上次跟进后 1 个月再次跟进。

当完成后重复还指定了星期几或每月某一天时，下一项任务会落在间隔最后一周（或一个月）中第一个匹配日期。“每 2 周的星期日”若在星期二完成，会在下下个星期日再次出现，而不是紧接着的星期日。

如果任务实际较早完成，但之后才标记为已完成，请改用实际完成时间（桌面端右键点击完成按钮；移动端长按状态菜单中的**已完成**），下一次任务便会从真正完成的时间开始安排。

## 结束重复

使用**结束**控制系列何时停止：

| 设置 | 含义 |
| --- | --- |
| **永不** | 每次完成后继续创建下一项任务。 |
| **指定日期** | 下一次发生时间超过所选日期时停止。 |
| **次数** | 达到所选发生次数后停止。 |

## 日历预览

一个重复系列只保留一项活跃任务。完成当前任务时，Mindwtr 才创建下一项真实任务。

如果希望提前规划下一次发生时间，请开启**在日历中显示下一次**。该预览有助于规划，但不是第二项活跃任务。

## 完成后会发生什么

完成重复任务后，Mindwtr 会创建下一项任务，保留相同的标题、项目、领域、情境、标签、备注、提醒和其他任务详情。下一次的检查清单项目会被重置。

如果任务有截止日期、开始日期或回顾日期，Mindwtr 会按重复规则推进这些字段。仅含日期的值仍保持仅含日期。

在日期到来前，下一项任务不会出现在专注和下一步行动中。未来开始日期会像往常一样推迟任务；没有开始日期的重复任务会推迟到下一个日期（截止日期和回顾日期中较早者）。这样，刚创建的下一项不会在你完成当前任务后立刻重新出现在列表里。已推迟的任务仍会计入“已隐藏（未来开始）”提示，你可以通过其**显示**开关查看，或在**项目**和**搜索**中找到它们。
