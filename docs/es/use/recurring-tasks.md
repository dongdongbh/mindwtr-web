# Tareas recurrentes

Usa tareas recurrentes para el trabajo que debe reaparecer después de completarlo: revisiones, labores domésticas, rutinas, renovaciones y mantenimiento.

La recurrencia se configura en el editor de tareas. Si usas Captura rápida, crea primero la tarea y ábrela después para editarla.

## Configurar una tarea recurrente

1. Abre la tarea.
2. Edítala.
3. Busca el campo **Recurrencia** en la sección de planificación.
4. Elige **Diaria**, **Semanal**, **Mensual** o **Anual**.
5. Ajusta los detalles de repetición.
6. Guarda la tarea.

Los controles de recurrencia aparecen en la pantalla de edición tanto en escritorio como en dispositivos móviles.

## Opciones de repetición

| Opción | Para qué usarla |
| --- | --- |
| **Diaria** | Hábitos y rutinas diarias. |
| **Semanal** | Revisiones semanales, reuniones, recados o mantenimiento. |
| **Mensual** | Facturas, revisiones mensuales, suscripciones o listas mensuales. |
| **Anual** | Renovaciones, impuestos, registros y aniversarios. |

Cada tipo permite usar **Repetir cada** para saltar intervalos, por ejemplo, cada 2 semanas o cada 3 meses.

Las tareas semanales pueden elegir días concretos. Las mensuales pueden repetirse el mismo día de cada mes o usar **Personalizada** para patrones como un día de la semana específico.

## Calendario fijo o después de completar

De forma predeterminada, Mindwtr mantiene un calendario fijo. Úsalo para plazos o compromisos que deben seguir vinculados al calendario.

Ejemplos:

- Pagar el alquiler cada mes.
- Revisar las finanzas el primer sábado.
- Renovar el seguro cada año.

Activa **Repetir después de completar** si la siguiente tarea debe calcularse según cuándo termines la actual.

Ejemplos:

- Regar las plantas 3 días después de la última vez.
- Limpiar los filtros 2 semanas después de completarlo.
- Hacer seguimiento 1 mes después del seguimiento anterior.

Si una repetición posterior a la finalización también selecciona días de la semana o del mes, la tarea siguiente cae en el primer día coincidente de la última semana o el último mes del intervalo. Si completas un martes una tarea que se repite «cada 2 semanas el domingo», volverá el domingo de la semana siguiente, no el domingo inmediato.

Si terminaste antes una tarea pero solo la marcas como Terminada más tarde, complétala usando la hora real —clic derecho en el botón de completar en escritorio o pulsación larga en **Terminada** en el menú de estado móvil— y la próxima aparición se planificará desde el momento en que la terminaste.

## Finalizar una recurrencia

Usa **Termina** para controlar cuándo se detiene la serie:

| Ajuste | Significado |
| --- | --- |
| **Nunca** | Sigue creando la siguiente tarea después de cada finalización. |
| **En una fecha** | Se detiene cuando la siguiente aparición superaría la fecha seleccionada. |
| **Después de** | Se detiene tras el número de apariciones seleccionado. |

## Vista previa en el calendario

Mindwtr mantiene una sola tarea activa por serie recurrente. Crea la siguiente tarea real cuando completas la actual.

Activa **Mostrar la próxima aparición en el Calendario** si quieres verla como previsión. Sirve para planificar, pero no es una segunda tarea activa.

## Qué ocurre al completarla

Al completar una tarea recurrente, Mindwtr crea la siguiente con el mismo título, proyecto, área, contextos, etiquetas, notas, recordatorios y demás detalles. Los elementos de su lista de comprobación se restablecen.

Si tiene fecha límite, fecha de inicio o fecha de revisión, Mindwtr hace avanzar esos campos según la regla de recurrencia. Los valores que solo contienen una fecha siguen sin hora.

La tarea siguiente no aparece en Foco ni en Próximas acciones hasta que llega su fecha. Una fecha de inicio futura la aplaza como de costumbre; una tarea recurrente sin fecha de inicio queda aplazada hasta su próxima fecha, la primera entre la fecha límite y la de revisión. Así, la nueva aparición no vuelve a la lista justo después de completar la actual. Las apariciones aplazadas siguen contando en el aviso «ocultas (inicio futuro)» y puedes revelarlas con **Mostrar**, o encontrarlas en **Proyectos** y **Buscar**.
