# Import from OmniFocus

Move an OmniFocus export into Mindwtr while preserving the intent of projects, contexts, and next actions where possible.

## Supported input

Mindwtr supports OmniFocus exports when the current app build includes the importer for the chosen export format.

## What imports preserve

- Task title and notes
- Project grouping
- Dates where available
- Completion state where available
- Context-like metadata where it maps cleanly

## After import

OmniFocus setups can be highly customized. After import, review every active project and confirm the next action is still clear.

## Good migration workflow

1. Export from OmniFocus.
2. Export a Mindwtr backup.
3. Run the import.
4. Check active projects first.
5. Rebuild Focus from trusted next actions.
