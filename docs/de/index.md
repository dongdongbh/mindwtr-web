## Was ist Mindwtr?

Mindwtr ist eine kostenlose, quelloffene Aufgabenverwaltung auf Grundlage von Getting Things Done (GTD). Sie funktioniert offline, benötigt kein Konto und speichert ihre primären Daten auf Ihrem Gerät. Die optionale Synchronisierung bleibt über iCloud, Dateien, Dropbox, WebDAV oder einen selbst gehosteten Server unter Ihrer Kontrolle.

Erfahren Sie mehr über den [Funktionsumfang von Mindwtr](/de/start/what-mindwtr-is) und die [Zusagen des Projekts](/de/start/commitments), oder entwickeln Sie mit der [lokalen API](/de/power-users/local-api) und dem [MCP-Server](/de/power-users/mcp).
