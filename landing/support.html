Support

Need help with Mindwtr?

Start here. The best way to get help depends on what you need — bugs and feature requests are easiest to track on GitHub, and private questions can go by email. Want to give back instead? There are plenty of free ways to help, too.

Get help

Report a bug or ask a question.

The best place to report a bug or ask for help is a GitHub issue — it keeps logs, device details, and follow-up together in one trackable place, so nothing gets lost. Please include your platform and app version (Settings → About) so it's easier to help.

  • Open a GitHub issue

    Best for bugs and feature requests — public, searchable, and trackable, so nothing slips.

    Open an issue →

  • Send in-app feedback

    Already in the app? Use Send feedback on the About page (Settings → About) — no GitHub account needed.

  • Email support

    For anything that doesn't fit a public issue, such as a private or account question.

    support@mindwtr.app

Ways to help

Most ways to help are free.

Can't or don't want to donate? No problem. These help just as much.

  • Star on GitHub

    Stars help people discover Mindwtr and signal the project is alive and maintained.

    Star the repo →

  • Help translate

    Mindwtr speaks 18 language options thanks to contributors. Add or improve your locale.

    Translation guide →

  • Leave a store review

    A rating on your app store helps more people find Mindwtr.

    App Store Google Play Microsoft Store

  • Share Mindwtr

    Tell a friend or your community. Word of mouth is how most people find it.

    Share on X Reddit

  • Report bugs & ideas

    Found something, or want a feature? Open an issue so it stays trackable.

    Open an issue →
