# 桌面版鍵盤快速鍵

Mindwtr 桌面版支援鍵盤優先操作，提供標準（Gmail/Todoist 風格）、Vim 與 Emacs 預設配置。在應用程式中按 `?`，即可開啟目前所用配置的快速鍵說明。

## 快速上手

- 在**設定 -> 一般 -> 鍵盤快速鍵**中選擇預設配置。
- 在應用程式中按 `?` 開啟目前配置的快速鍵表。表格末尾有快速新增語法表，列出所有符記（`/start:`、`/note:`、`@context`、`+Project`……）。
- 按 `/` 搜尋。
- 標準與 Vim 模式可用 `gi` 前往收集箱、`gn` 前往下一步行動、`gf` 前往專注。
- Mindwtr 取得焦點時，按 `a` 新增任務；同一個應用程式內新增任務動作仍可使用 `o`。
- 用 `j`、`k` 移動選取項目，按 `Enter` 開啟所選任務。
- 標準模式中，按 `e` 標示完成、`x` 選取任務以進行批次操作、`Shift+Enter` 編輯、`#` 刪除、`z` 復原。
- Vim 模式中，按 `e` 編輯、`x` 切換完成狀態、`dd` 刪除。
- 無論使用哪種配置，皆可按 `Ctrl+Z` / `Cmd+Z` 復原最近一次完成或刪除任務的動作。
- 無論使用哪種配置，先按 `s` 再按一個字母，即可設定所選任務的狀態：`si` 收集箱、`sn` 下一步、`sw` 等待中、`ss` 將來、`sd` 已完成、`sa` 已歸檔。提示訊息會確認變更並提供復原按鈕。
- 按 `Insert` 跳至新增任務輸入欄（在沒有輸入欄的檢視中會開啟快速新增）。
- 在任何文字欄位之外直接按 `1`-`9`，即可依側邊欄順序切換領域；按 `0` 則清除領域篩選。原本的 `Shift+A` 加數字組合仍然有效。單按 `a` 會開啟快速新增。
- 桌面版按 `Ctrl+Alt+S` 執行手動同步。
- 在右鍵 / `Shift+F10` 情境選單中：↑/↓ 在項目間移動、→ 開啟子選單面板、← 返回、Enter 啟用、Esc 關閉並將焦點還給任務。

在可編輯欄位中輸入時，若快速鍵會與文字輸入衝突，便會忽略該快速鍵。

桌面版指南是目前快速鍵表、快速新增自訂、鍵盤行為注意事項與邊界情況的標準參考文件。

## 全域快速新增

可在**設定 -> 一般 -> 輸入**中變更或停用全域「快速新增」快速鍵。全域快速鍵是否可用，取決於桌面平台權限及封裝方式。

全域「快速新增」與應用程式取得焦點時使用的 `a` 不同。全域快速鍵可從其他應用程式觸發；`a` 只有在 Mindwtr 本身取得焦點，而且你未在文字欄位中輸入時才有效。
