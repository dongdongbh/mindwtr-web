GTD, de principio a fin

Capturar, aclarar, organizar, reflexionar, actuar. Mindwtr mantiene los pasos conectados en lugar de dejar que se conviertan en tareas sueltas. ¿GTD es nuevo para ti? Empieza aquí →

  • Capturar →

    Añade rápido desde cualquier sitio: atajo global, bandeja del sistema, hoja de compartir o voz.

  • Aclarar →

    Procesado guiado de la bandeja de entrada con la regla de los dos minutos.

  • Organizar →

    Proyectos, secciones, contextos y listas por estado.

  • Reflexionar →

    Revisiones diarias y semanales con recordatorios.

  • Actuar →

    Siguientes acciones filtradas por contexto, para que siempre sepas qué toca.

Una vista para cada parte del sistema

Cada vista está construida para el método GTD, no añadida a posteriori.

  • Enfoque →

    Tu agenda del día y tus siguientes acciones, juntas en una sola pantalla.

  • Proyectos →

    Objetivos de varios pasos con secciones, áreas y orden manual de tareas.

  • Calendario →

    Planifica el trabajo con fecha y suscríbete a calendarios externos y feeds ICS.

  • Tablero →

    Estilo kanban: arrastra tareas entre Bandeja de entrada, Siguiente, En espera y Algún día.

  • Contextos y etiquetas →

    Etiqueta las tareas según dónde las haces, como @casa o @trabajo/reuniones; los contextos anidados cuentan con su contexto padre.

  • Revisión →

    Asistentes de revisión diaria y semanal que mantienen el sistema fiable.

Profundidad cuando la buscas

Potencia que no estorba hasta que la necesitas.

  • Búsqueda global →

    Operadores como status:, context: y due:<=7d en todo tu sistema.

  • Captura por voz →

    Di una tarea en voz alta; se transcribe y se guarda sola en tu bandeja de entrada.

  • Enfoque Pomodoro →

    Temporizadores de 15/3, 25/5 y 50/10, más uno personalizado, dentro de Enfoque.

  • Listas reutilizables →

    Duplica tareas o reinicia listas de comprobación para las rutinas que repites una y otra vez.

  • Adjuntos y notas →

    Archivos, enlaces y notas Markdown con vista previa en cualquier tarea.

  • Repetición flexible →

    La próxima fecha se calcula cuando terminas, no según un calendario fijo.

Tus datos, a tu manera

Tus tareas se quedan en tu dispositivo. La sincronización es opcional, y tú eliges dónde. Sin cuenta.

  • En tu dispositivo →

    Tareas, notas, proyectos y adjuntos empiezan en tu dispositivo (local primero).

  • Sincronización flexible →

    iCloud/CloudKit nativo en dispositivos Apple, más sincronización de archivos, WebDAV, Dropbox y nube autoalojada.

  • Importar →

    Trae tus tareas desde Todoist, TickTick, DGT GTD u OmniFocus.

  • Obsidian →

    Importa un vault con enlaces profundos de vuelta a las notas originales (escritorio).

  • Calendarios externos →

    Lee los calendarios del sistema y de Apple, envía tareas con fecha y suscríbete a ICS.

  • Copia de seguridad y restauración →

    Exporta a JSON y restaura desde una copia de seguridad validada de Mindwtr.

Potencia opcional para los curiosos

La IA y la automatización siguen siendo opcionales: actívalas solo si las quieres.

  • Asistente de IA →

    Conecta tu propia cuenta de OpenAI, Gemini o Claude, o ejecuta un modelo local o autoalojado.

  • API REST local →

    Una API en localhost autenticada con token para programar la app de escritorio.

  • Servidor MCP →

    Maneja Mindwtr desde clientes LLM con el servidor publicado mindwtr-mcp.

  • Atajos de Apple y Siri →

    Automatiza la captura y las acciones en dispositivos Apple.

  • App web (PWA) →

    Acceso desde el navegador, también sin conexión.

  • Docker →

    Ejecuta la PWA y el servidor de sincronización autoalojado con Docker.
