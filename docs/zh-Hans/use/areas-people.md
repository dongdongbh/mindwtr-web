# 领域与人员

使用**领域**表示持续责任，使用**人员**表示委派或以人为中心的工作。两者都可在分配时创建，也可之后在设置中管理。

## 领域

领域把相关项目和独立任务归入稳定的责任范围，例如工作、家庭、健康、研究或客户名称。

当你想回答“这项工作属于我生活或工作的哪一部分？”时，请使用领域。

### 分配时创建领域

创建领域无需离开当前任务流程。

1. 打开快速添加、快速记录或任务编辑器。
2. 打开**领域**字段。
3. 输入领域名称。
4. 如果尚不存在，选择**创建**。
5. 新领域会被选中并分配给任务。

桌面端的领域选择器也支持边输入边创建。移动端的快速记录和任务编辑领域选择器允许先搜索，没有完全匹配项时再创建领域。

### 在设置中创建或编辑领域

要直接管理已保存领域，请使用设置。

1. 打开**设置**。
2. 打开**管理**。
3. 展开**领域**。
4. 添加新领域、重命名、更改颜色，或删除不再使用的领域。

删除领域会保留其中的任务和项目。Mindwtr 会将它们改为未分配，而不会删除工作。

## 人员

人员是可重复使用的受托人名称，用于委派任务、等待中事项和以人为中心的搜索。人员还可保存备注与参考链接，例如电子邮件地址、联系人页面或 Obsidian 链接。

当你想回答“这件事由谁负责、我在等谁，或应该和谁讨论？”时，请使用人员。

### 分配时创建人员

1. 打开任务。
2. 找到**分配给**或**受托人/人员**。
3. 输入人员姓名。
4. 如果尚未保存，选择**新建人员**。
5. 系统会保存该人员并分配给任务。

即使不创建已保存人员，直接输入名称也可作为任务受托人。选择**新建人员**后，该名称会在今后的建议和设置中重复使用。

### 记录时分配人员

在快速添加中输入 `%` 加姓名：`Ask %Jim for the budget /waiting`。已知名称无需引号即可匹配（包括多词名称）；新的多词名称请写作 `%"Jim Smith"`。桌面端输入 `%` 会打开与“分配给”字段相同的人员建议。

### 在设置中创建或编辑人员

1. 打开**设置**。
2. 打开**管理**。
3. 展开**人员**。
4. 添加人员、重命名、添加备注、添加参考链接，或删除不再使用的人员。

删除人员会保留任务。Mindwtr 只清除已保存的人员记录，不会删除工作。

## 相关字段

| 字段 | 用途 | 创建位置 |
| --- | --- | --- |
| **领域** | 项目与任务的责任分类 | 领域选择器、设置 -> 管理 -> 领域 |
| **人员** | 委派受托人或等待事项负责人 | 快速添加中输入 `%person`、分配给字段、设置 -> 管理 -> 人员 |
| **情境** | 你能在何处、使用何种工具或与谁一起完成任务 | 快速添加或任务编辑中输入 `@context` |
| **标签** | 主题、精力、模式或分组标记 | 快速添加或任务编辑中输入 `#tag` |
