Mindwtr 的中文名稱是「如水」。

## Mindwtr 是什麼？

Mindwtr 是一款以 Getting Things Done（GTD）為核心的免費開放原始碼任務管理工具。它可離線運作、不需要帳號，主要資料都保存在你的裝置上。你也可以自行掌控選用的同步方式，包括 iCloud、檔案、Dropbox、WebDAV 或自行託管的伺服器。

進一步瞭解[Mindwtr 的定位](/zh-Hant/start/what-mindwtr-is)與[我們的承諾](/zh-Hant/start/commitments)，或透過[本機 API](/zh-Hant/power-users/local-api) 與 [MCP 伺服器](/zh-Hant/power-users/mcp)進行開發。
