# 重複任務

需要在完成後再次出現的工作，例如回顧、家務、例行工作、續約與維護，請使用重複任務。

重複規則位於任務編輯器。若使用快速收集，請先建立任務，再開啟編輯。

## 設定重複任務

1. 開啟任務。
2. 編輯任務。
3. 在排程區段找到**重複**欄位。
4. 選擇**每天**、**每週**、**每月**或**每年**。
5. 調整重複細節。
6. 儲存任務。

桌面版與行動版任務編輯畫面都提供這些控制項。

## 重複選項

| 選項 | 適用情況 |
| --- | --- |
| **每天** | 習慣與每日例行事項。 |
| **每週** | 每週回顧、會議、跑腿或維護。 |
| **每月** | 帳單、每月回顧、訂閱或每月檢查清單。 |
| **每年** | 年度續約、稅務、登記與週年紀念。 |

每種重複類型都可用**每隔**跳過間隔，例如每 2 週或每 3 個月。

每週任務可選特定星期；每月任務可在每月同一天重複，也可用**自訂**設定特定星期等模式。

## 固定排程或完成後重複

如水預設維持固定排程，適合必須綁定行事曆的截止日期或承諾。

例如：

- 每月繳房租。
- 每月第一個星期六檢視財務。
- 每年續保。

若下一項任務應根據目前任務的完成時間計算，請開啟**完成後重複**。

例如：

- 上次澆水 3 天後再次澆水。
- 完成後 2 週清潔濾網。
- 上次追蹤 1 個月後再次追蹤。

若「完成後重複」也指定星期或每月日期，下一項任務會落在間隔最後一週（或一個月）中第一個相符日期。「每 2 週星期日」的任務若在星期二完成，會於下下個星期日再出現，而不是即將到來的星期日。

若任務其實較早完成，只是稍後才標示「已完成」，請改以實際時間完成（桌面版在完成按鈕上按右鍵；行動版長按狀態選單中的**已完成**）。下一次排程會從實際完成時間計算。

## 結束重複

使用**結束**控制系列何時停止：

| 設定 | 意義 |
| --- | --- |
| **永不** | 每次完成後持續建立下一項任務。 |
| **於日期** | 下一次發生時間超過所選日期時停止。 |
| **次數後** | 達到所選發生次數後停止。 |

## 行事曆預覽

如水只為重複系列保留一項使用中的任務；完成目前任務後，才建立下一項真實任務。

若想預先規劃下一次發生時間，請開啟**在行事曆顯示下一次發生時間**。此預覽協助規劃，但不是第二項使用中任務。

## 完成後會發生什麼

完成重複任務時，如水會建立下一項任務，保留相同標題、專案、領域、情境、標籤、筆記、提醒與其他任務細節，並重設檢查清單項目。

若任務有截止日期、開始日期或回顧日期，如水會依重複規則推進這些欄位；只有日期的值仍保持只有日期。

下一項任務在日期到來前不會進入「專注」與「下一步行動」。未來開始日期會照常將它延後；沒有開始日期的重複任務則延後至下一日期（截止與回顧日期中較早者）。如此可避免剛建立的下一次任務在完成目前任務後立刻重回清單。延後的項目仍計入「已隱藏（未來開始）」通知；你可用其中的**顯示**切換鈕揭露，或在**專案**與**搜尋**中找到它們。
