# Import from DGT GTD

Use the DGT GTD importer when you have an export file from DGT GTD and want to preserve as much structure as possible.

## Supported data

Mindwtr maps source data where it has a clear GTD equivalent:

- Task titles and notes.
- Status or completion state.
- Projects or folders when they map cleanly.
- Contexts and tags when present.
- Dates when the export provides them.

## Import flow

1. Export from DGT GTD.
2. Export a Mindwtr backup.
3. Run the importer.
4. Review imported projects and contexts.
5. Move only real next actions into active use.

## After import

Imported systems usually contain old commitments. Review before trusting the result. Archive or delete stale material instead of trying to perfectly recreate the old app.
