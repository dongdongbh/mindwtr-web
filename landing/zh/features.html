GTD，从头到尾

收集、理清、组织、回顾、执行。Mindwtr 让各个步骤环环相扣，而不是变成一件件孤立的杂务。刚接触 GTD？从这里开始 →

  • 收集 →

    随时随地快速添加——全局快捷键、托盘、分享菜单或语音。

  • 理清 →

    引导式清理收集箱，内置两分钟法则。

  • 组织 →

    项目、分组、情境和状态清单。

  • 回顾 →

    每日和每周回顾流程，带提醒。

  • 执行 →

    按情境筛选的下一步行动，让你永远知道接下来做什么。

系统的每个部分都有对应的视图

每个视图都为 GTD 方法而设计，而不是事后拼凑。

  • 专注 →

    按时间排列的日程和你的下一步行动，同屏呈现。

  • 项目 →

    多步骤目标，支持分组、领域和手动排序。

  • 日历 →

    安排有日期的工作，并订阅外部日历和 ICS 日程源。

  • 看板 →

    看板式拖放，在收集箱、下一步、等待中、将来/也许之间移动任务。

  • 情境与标签 →

    按做事的场合给任务打标签，比如 @家里 或 @工作/会议；嵌套情境会归入上级情境。

  • 回顾 →

    每日和每周回顾向导，让系统始终可信。

深度功能，需要时才出现

强大的功能平时不打扰你，需要时才登场。

  • 全局搜索 →

    支持 status:、context:、due:<=7d 等运算符，搜遍所有内容。

  • 语音记录 →

    说出一个任务，它会被自动转成文字并放进你的收集箱。

  • 番茄钟专注 →

    专注视图内置 15/3、25/5、50/10 计时器，还可以自定义一组。

  • 可复用清单 →

    复制任务或重置检查清单，应对一遍遍重复的日常流程。

  • 附件与笔记 →

    任何任务都能附上文件、链接和支持预览的 Markdown 笔记。

  • 弹性重复 →

    下一次日期在你完成后才计算，而不是按死板的固定周期。

你的数据，按你的方式

任务留在你的设备上。同步是可选的，同步到哪里由你决定。无需账号。

  • 在你的设备上 →

    任务、笔记、项目和附件都从你的设备开始（本地优先）。

  • 灵活同步 →

    Apple 设备上原生 iCloud/CloudKit，另有文件同步、WebDAV、Dropbox 和自建云端。

  • 导入 →

    从 Todoist、TickTick、DGT GTD 或 OmniFocus 迁移你的任务。

  • Obsidian →

    导入仓库，并保留跳回原笔记的深层链接（桌面端）。

  • 外部日历 →

    读取系统和 Apple 日历，推送有日期的任务，并订阅 ICS。

  • 备份与恢复 →

    导出为 JSON，并从经过校验的 Mindwtr 备份恢复。

进阶能力，留给爱折腾的人

AI 和自动化始终是可选项——想用再启用。

  • AI 助手 →

    接入你自己的 OpenAI、Gemini 或 Claude 账号，或运行本地/自建模型。

  • 本地 REST API →

    基于令牌认证的 localhost API，用脚本控制桌面应用。

  • MCP 服务器 →

    通过已发布的 mindwtr-mcp 服务器，从 LLM 客户端操控 Mindwtr。

  • Apple 快捷指令与 Siri →

    在 Apple 设备上自动化收集和各种操作。

  • 网页应用（PWA） →

    浏览器直接访问，支持离线。

  • Docker →

    用 Docker 运行 PWA 和自建同步服务器。

在哪工作都是原生体验

  • 桌面端 →

    macOS、Windows 和 Linux。

  • 移动端 →

    iOS 和 Android，外加 Android 主屏幕专注小组件。

  • 键盘 →

    Vim 和 Emacs 键位预设，纯键盘高效操作。

  • 主题 →

    浅色与深色，跟随系统。

  • 18 种语言 →
