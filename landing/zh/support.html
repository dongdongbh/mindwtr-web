支持

需要 Mindwtr 的帮助吗？

从这里开始。获取帮助的最佳方式取决于你的需要——问题和功能建议最适合在 GitHub 上跟进，私人问题可以发邮件。想为项目出份力？也有很多免费的方式。

获取帮助

报告问题或提出疑问。

报告问题或寻求帮助的最佳去处是 GitHub issue——日志、设备信息和后续跟进都集中在一个可追踪的地方，不会丢失。请附上你的平台和应用版本（设置 → 关于），这样更容易帮到你。

  • 提交 GitHub issue

    最适合报告问题和提功能建议——公开、可搜索、可追踪，不会有遗漏。

    提交 issue →

  • 从应用内发送反馈

    已经在用应用了？在“关于”页面（设置 → 关于）使用发送反馈——无需 GitHub 账号。

  • 邮件支持

    不适合公开讨论的事情，比如私人问题，可以发邮件。

    support@mindwtr.app
