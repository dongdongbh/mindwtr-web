# WebDAV 同步

WebDAV 同步適合想採用標準、自行掌控的儲存端點，又不想執行 Mindwtr Cloud 伺服器的使用者。

## 適用時機

符合以下情況時可使用 WebDAV：

- 已經有 WebDAV 伺服器
- 需要跨平台同步
- 偏好自行掌控的基礎設施
- 熟悉伺服器 URL 與憑證管理

## 設定檢查清單

1. 匯出備份。
2. 在 WebDAV 伺服器上建立如水專用資料夾。
3. 在如水中輸入伺服器 URL 與憑證。
4. 在第一部裝置上執行手動同步。
5. 再連接一部裝置，確認測試任務能雙向同步。

::: tip 使用專用資料夾
請保留一個只存放如水同步資料的資料夾，不要將應用程式同步檔案與無關文件混在一起。
:::

## 疑難排解

- 確認 URL 指向可寫入的資料夾。
- 檢查伺服器憑證是否有效。
- 確認具備上傳與覆寫權限。
- 留意服務供應商的速率限制或檔案鎖定。
- 刪除伺服器端檔案前先匯出備份。
