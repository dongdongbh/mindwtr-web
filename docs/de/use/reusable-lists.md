# Wiederverwendbare Listen (Vorlagen)

Mindwtr unterstützt wiederverwendbare Checklisten für Aufgaben, die sich ohne festen Zeitplan wiederholen, etwa Packlisten, Einkäufe oder Reisevorbereitungen.

## Zwei Vorgehensweisen

### Duplizieren (Vorlagen)

Verwenden Sie diese Variante, wenn jeder Durchlauf eine neue Kopie sein soll.

Beispiel: **Packliste**

1. Bewahren Sie eine Master-Aufgabe unter Irgendwann/Vielleicht auf.
2. Duplizieren Sie sie bei der Planung einer Reise.
3. Bearbeiten Sie die Kopie, ohne das Original zu verändern.

Beim Duplizieren wird Folgendes zurückgesetzt:

- Checklistenpunkte beginnen unabgehakt.
- Planungs- und Wiedervorlagedaten werden gelöscht.
- Die Wiederholung wird entfernt.
- Die Kopie wird zu einer neuen Aufgabe mit neuen Checklisten- und Anhangs-IDs.

### Checkliste zurücksetzen

Verwenden Sie diese Variante, wenn Sie dieselbe Liste erneut brauchen.

Beispiel: **Einkauf**

1. Haken Sie beim Einkaufen die Einträge ab.
2. Tippen Sie auf **Checkliste zurücksetzen**.
3. Verwenden Sie die Liste beim nächsten Mal erneut.

Das Zurücksetzen:

- entfernt alle Häkchen,
- aktualisiert die vorhandene Aufgabe, statt eine zweite Kopie anzulegen, und
- öffnet die Aufgabe erneut, falls sie bereits als erledigt markiert war.

## Wo die Funktionen zu finden sind

- **Mobilgeräte:** Aufgaben-Editor → **Aufgabe duplizieren** oder **Checkliste zurücksetzen**
- **Desktop:** Aktionsmenü der Aufgabe

## Welche Variante passt?

- Verwenden Sie **Aufgabe duplizieren** für Reisevorbereitungen, Einführungschecklisten und alles, bei dem jeder Durchlauf eigene Notizen und Termine erhalten soll.
- Verwenden Sie **Checkliste zurücksetzen** für Einkäufe, wiederkehrende Wartung und andere Listen, bei denen dieselbe Aufgabe langfristig nützlich bleibt.
- Enthält die Liste außerdem Termine, Wiederholungen oder projektspezifische Notizen, ist Duplizieren meist sicherer.

## Tipps

- Bewahren Sie die Master-Version unter **Irgendwann/Vielleicht** oder in einem unauffälligen Projekt auf, damit sie die täglichen Listen nicht überlädt.
- Geben Sie häufig duplizierten Vorlagen eindeutige Namen wie `Packliste (Vorlage)`.
- Wenn einzelne Checklistenpunkte eigene Fälligkeiten oder Wiederholungsregeln benötigen, teilen Sie die Liste in separate Aufgaben auf, statt eine wiederverwendbare Liste zu überfrachten.
