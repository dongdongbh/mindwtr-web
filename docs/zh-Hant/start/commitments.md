# 我們的承諾

軟體專案會終止，公司會被收購，資金會耗盡，應用程式也可能停止服務。當任務系統保存了你多年的承諾，在依賴它之前，這項風險值得仔細考量。本頁列出保護你的各項條件；每一項都是今天就能查證的事實，而不是對未來的承諾。

## 授權確保每個版本都能繼續使用

如水依 [AGPL-3.0](https://www.gnu.org/licenses/agpl-3.0.html) 授權釋出。每個已發行版本都會永遠適用這些條款。不論專案日後如何，你電腦上的版本仍可運作，原始碼也持續開放，任何人都能建置、分支或維護。

## 不必依賴帳號

核心工作流程在你的裝置上執行，不需要帳號。你無須註冊，也不必先通過登入畫面，便能收集、釐清與回顧。沒有任何由如水營運的服務保存你的資料，因此不會有某部伺服器關閉後讓你無法存取資料。

## 你的資料是可自行讀取的一般檔案

如水在你的裝置上以 SQLite 與 JSON 儲存資料。匯出與備份是內建功能，而且會一直保留：如水沒有付費層級，也就不會把它們鎖在付費牆後。[資料庫結構描述](/zh-Hant/developers/database-schema)有完整文件，因此格式維持開放而非封閉專有。在可行的情況下，儲存格式變更會隨附遷移程序，升級時一併帶著你的資料前進。

## 同步方式由你選擇

你可以決定如水是否同步，以及透過哪種方式同步：完全不同步、iCloud、Dropbox、WebDAV，或自行託管的伺服器。無論選擇哪一種，資料只會經過你已掌控的基礎設施；中間沒有如水營運的服務保存副本。請參閱[資料與同步](/zh-Hant/data-sync/)瞭解各選項的運作方式。

## AI 在你啟用前保持關閉

AI 助理是選用功能，預設為關閉。它使用你自己的金鑰與服務供應商；供應商也可以是 llama.cpp 或 Ollama 等本機端點，讓資料完全不必離開電腦。請參閱 [AI 助理](/zh-Hant/power-users/ai-assistant)。

## 架構本身就是保障

訂閱制筆記服務要求你相信背後的公司會保持健全，也能持續支付營運開銷。如水所要求的信任更少，因為它所依賴的條件更少。檔案屬於你、格式已有文件、授權無法撤銷，核心循環也不需要伺服器。即使開發明天停止，你手上的版本仍會繼續運作。

如要深入瞭解如水如何處理資料，請閱讀[隱私權政策](https://mindwtr.app/privacy)。
