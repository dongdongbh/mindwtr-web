# Mobile guide

Mindwtr mobile is optimized for capture, quick decisions, and checking Focus away from your desk.

## Best uses on mobile

- Capture a task before it leaves your head
- Check Focus during the day
- Review waiting items while away from the desktop
- Add notes or attachments to an existing task
- Run a lightweight review when you have a quiet moment

## Capture quickly

Use Inbox for anything unfinished or unclear. Do not over-organize while capturing. If you know the project, context, tag, or due date, quick syntax works the same way as desktop.

| Syntax | Use |
| --- | --- |
| `@home` | Add a context |
| `#travel` | Add a tag |
| `+Trip` | Assign a project |
| `/due:tomorrow` | Add a due date |
| `/next` | Mark as next action |

## Work from Focus

Focus keeps the mobile app useful without forcing you through every list. Start there when you want to know what matters today.

## Sync expectations

Mobile sync depends on the backend you choose and the operating system's background behavior. If a change does not appear immediately on another device, open Mindwtr on both devices and run a manual sync.

::: warning Before restore
Export a backup before restoring, importing, or switching sync backends. Restore operations intentionally replace local data.
:::

## Notifications and widgets

Supported mobile builds can show reminders and widget surfaces. If reminders do not fire, check system notification permission, battery restrictions, and the task's date/time settings.
