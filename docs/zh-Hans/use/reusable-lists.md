# 可重复使用的清单（模板）

如水支持可重复使用的检查清单，适合没有严格日程的重复事项（如打包、购物、出行准备）。

## 两种模式

### 复制（模板）

每次都需要一份全新副本时使用。

示例：**打包清单**

1. 在“将来/也许”中保留一项主任务
2. 计划出行时复制它
3. 编辑副本，不影响主任务

复制时会重置：

- 检查清单项目恢复为未勾选
- 清除日程和回顾日期
- 清除重复规则
- 副本成为一项新任务，拥有新的检查清单与附件 ID

### 重置检查清单

想再次使用同一份清单时使用。

示例：**购物清单**

1. 购物时勾选项目
2. 点按**重置检查清单**
3. 下次继续使用

重置会：

- 取消勾选所有检查清单项目
- 就地更新任务，而不是创建第二份副本
- 如果任务已标记完成，则重新打开它

## 在哪里找到

- **移动端：**任务编辑器 → **复制任务**或**重置检查清单**
- **桌面端：**任务操作菜单

## 应该选择哪一种

- 出行准备、入门检查清单，或每次都应拥有独立笔记和时间安排的事项，请使用**复制任务**。
- 购物、周期维护，或同一项任务长期都能继续使用的清单，请使用**重置检查清单**。
- 如果清单还带有日期、重复规则或项目专属笔记，复制通常更稳妥。

## 提示

- 将主版本保存在**将来/也许**或干扰较少的项目中，以免占据日常清单。
- 如果经常复制模板式任务，可使用 `打包清单（模板）` 之类一目了然的名称。
- 如果检查清单中的每一项都开始需要截止日期或重复规则，请拆分为独立任务，不要让一份可重复使用的清单承载过多功能。
