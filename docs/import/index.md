## Before import

1. Export your source app data.
2. Export a Mindwtr backup.
3. Import into Mindwtr.
4. Review imported projects, contexts, and next actions.
5. Delete noise before enabling sync broadly.

## Other migration paths

For apps without a native importer, use plain text, CSV-to-quick-add scripts, the local API, or MCP. Keep the migration boring: import first, clarify later.
