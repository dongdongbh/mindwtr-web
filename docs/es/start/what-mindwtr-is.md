# Qué es Mindwtr

Mindwtr gestiona los compromisos y las acciones de tu vida personal mediante el método Getting Things Done de David Allen. Esta página describe el centro del producto para que puedas decidir si encaja con tu forma de trabajar.

## GTD desde su concepción

Bandeja de entrada, Proyectos, Contextos, En espera, Algún día y Revisión son partes esenciales de Mindwtr. La aplicación está construida en torno a ellas; no son etiquetas añadidas a una lista de tareas genérica. Al capturar algo, aclararlo y archivarlo, recorres directamente el ciclo GTD. La [introducción a GTD](/es/use/gtd-overview) explica el método y el [flujo de trabajo GTD](/es/use/gtd-workflow) muestra cómo se corresponde cada paso con una pantalla.

## Las notas respaldan el trabajo

Puedes adjuntar notas y archivos a las tareas y los proyectos para tener junto al trabajo el contexto que necesitas. Mindwtr se detiene ahí. No es una base de conocimiento ni pretende convertirse en una. Cuando tu material de referencia necesita un lugar propio, consérvalo en una herramienta creada para ello. La [integración con Obsidian](/es/power-users/obsidian) existe precisamente por este motivo: tus notas permanecen en tu bóveda y Mindwtr extrae de ellas las tareas.

## Personal, no para equipos

Mindwtr guarda los compromisos de una persona: los tuyos. No hay espacios de trabajo compartidos, asignación de tareas a compañeros, hilos de comentarios ni un registro de actividad. En espera registra lo que otras personas te deben, como anotación personal y no como canal de colaboración.

## Sencillo de forma predeterminada y potente cuando hace falta

El ciclo diario consiste en capturar, aclarar y revisar, y todo lo necesario está visible sin configuración. Las funciones más avanzadas —IA, API local, servidor MCP, tableros y Pomodoro— no estorban hasta que las activas, y activarlas nunca dificulta el ciclo básico. El README del proyecto lo expresa con claridad:

> No me muestres una cabina de piloto cuando solo quiero montar en bicicleta.

## Mejor automático que manual

Cuando existe un valor predeterminado razonable, Mindwtr lo elige en lugar de preguntarte. Respeta las convenciones de la plataforma y el canal desde el que instalaste la aplicación, en vez de añadir un ajuste para cada decisión. Menos interruptores significan menos mantenimiento y menos posibilidades de error.
