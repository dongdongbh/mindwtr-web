# 桌面端键盘快捷键

Mindwtr 桌面端支持以键盘为主的操作，并提供标准（类似 Gmail/Todoist）、Vim 和 Emacs 三种预设。在应用中按 `?` 可打开当前预设的快捷键帮助。

## 快速入门

- 在**设置 -> 常规 -> 键盘快捷键**中选择预设。
- 在应用中按 `?` 打开当前预设的快捷键表。表格末尾还有快速添加语法，列出所有标记（`/start:`、`/note:`、`@context`、`+Project` 等）。
- 按 `/` 搜索。
- 在标准和 Vim 模式下，使用 `gi` 前往收集箱、`gn` 前往下一步行动、`gf` 前往专注。
- Mindwtr 获得焦点时，按 `a` 添加任务；`o` 仍可执行相同的应用内添加操作。
- 用 `j` 和 `k` 移动选择，按 `Enter` 打开所选任务。
- 在标准模式下，按 `e` 标记完成，按 `x` 选择任务以执行批量操作，按大写 `S` 将所选任务添加到今日专注或从中移除，按 `F2` 重命名，按 `Shift+Enter` 编辑，按 `#` 删除，按 `z` 撤销。
- 在 Vim 模式下，按 `e` 编辑，按 `x` 切换完成状态，按 `dd` 删除。
- 在任何预设中，使用 `Ctrl+Z` / `Cmd+Z` 撤销最近一次完成或删除任务。
- 在任何预设中，先按 `s` 再按一个字母来设置所选任务的状态：`si` 收集箱、`sn` 下一步、`sw` 等待中、`ss` 将来、`sd` 已完成、`sa` 已归档。提示消息会确认更改并提供撤销按钮。
- 按 `Insert` 跳到添加任务输入框（在没有输入框的视图中会打开快速添加）。
- 在任意文本框之外直接按 `1`-`9`，可按侧边栏顺序切换到某个领域；按 `0` 可清除领域筛选。原来的 `Shift+A` 加数字组合仍然有效。单按 `a` 会打开快速添加。
- 桌面端按 `Ctrl+Alt+S` 执行手动同步。
- 在右键 / `Shift+F10` 上下文菜单中：↑/↓ 在项目间移动，→ 打开子菜单面板，← 返回，Enter 激活，Esc 关闭并把焦点还给任务。

在可编辑字段中输入时，如果快捷键会与文字输入冲突，该快捷键会被忽略。

有关最新快捷键表、快速添加自定义、键盘行为说明和边界情况，请以桌面端指南为准。

## 全局快速添加

在**设置 -> 常规 -> 输入**中更改或禁用全局快速添加快捷键。全局快捷键依赖桌面平台权限和打包方式。

全局快速添加快捷键与应用获得焦点时的 `a` 快捷键不同。全局快捷键可以从其他应用触发；`a` 仅在 Mindwtr 本身获得焦点且你没有在文本字段中输入时有效。
