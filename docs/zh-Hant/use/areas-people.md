# 領域與人員

以**領域**管理持續承擔的責任，以**人員**管理委派或以特定人物為中心的工作。兩者都能在指派時建立，之後也能在設定中管理。

## 領域

領域依穩定的責任範圍，將相關專案與獨立任務歸在一起，例如工作、家庭、健康、研究或客戶名稱。

當你想回答「哪些工作屬於我生活或工作中的這個面向？」時，就使用領域。

### 指派時建立領域

你不必離開任務流程才能建立領域。

1. 開啟「快速新增」、「快速收集」或任務編輯器。
2. 開啟**領域**欄位。
3. 輸入領域名稱。
4. 若尚不存在，選擇**建立**。
5. 新領域會套用至該任務。

桌面版的領域選擇器同樣支援邊輸入邊建立。行動版「快速收集」與任務編輯中的領域選擇器，可先搜尋；找不到完全相符的項目時再建立領域。

### 在設定中建立或編輯領域

要直接管理已儲存的領域時，請使用設定。

1. 開啟**設定**。
2. 開啟**管理**。
3. 展開**領域**。
4. 新增領域，或重新命名、變更顏色、刪除不再使用的領域。

刪除領域不會刪除其中的任務與專案；如水只會將它們改成未指派。

## 人員

「人員」是可重複使用的受指派者名稱，適用於已委派任務、等待中項目，以及以人物為中心的搜尋。人員也可保存筆記與參考連結，例如電子郵件地址、聯絡人頁面或 Obsidian 連結。

當你想回答「這是誰負責的？我在等誰？或我該和誰討論？」時，就使用人員。

### 指派時建立人員

1. 開啟任務。
2. 找到**指派給**或**受指派者／人員**。
3. 輸入姓名。
4. 若尚未儲存，選擇**新增人員**。
5. 系統會儲存該人員並指派給任務。

即使不建立已儲存的人員，直接輸入名稱仍可作為任務的受指派者。選擇**新增人員**後，該名稱才會在日後建議與設定中重複使用。

### 收集時指派人員

在「快速新增」中輸入 `%` 加上姓名：`Ask %Jim for the budget /waiting`。已知名稱無須引號即可比對（包括多字名稱）；新建的多字名稱則寫成 `%"Jim Smith"`。桌面版輸入 `%` 時，會開啟與「指派給」欄位相同的人員建議。

### 在設定中建立或編輯人員

1. 開啟**設定**。
2. 開啟**管理**。
3. 展開**人員**。
4. 新增或重新命名人員、加入筆記或參考連結，或刪除不再使用的人員。

刪除人員不會刪除任務；如水只會清除該筆已儲存的人員記錄。

## 相關欄位

| 欄位 | 用途 | 建立位置 |
| --- | --- | --- |
| **領域** | 專案與任務所屬的責任範圍 | 領域選擇器、設定 -> 管理 -> 領域 |
| **人員** | 已委派工作的受指派者或等待對象 | 在快速新增輸入 `%person`、指派給欄位、設定 -> 管理 -> 人員 |
| **情境** | 可在何處、使用何種工具，或與誰一起完成任務 | 在快速新增或任務編輯中輸入 `@context` |
| **標籤** | 主題、精力、模式或分組標記 | 在快速新增或任務編輯中輸入 `#tag` |
