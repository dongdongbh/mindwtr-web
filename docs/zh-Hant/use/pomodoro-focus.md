# 番茄鐘專注

Mindwtr 的桌面版與行動版在**專注**檢視中提供選用的番茄鐘面板。

為了讓「專注」保持精簡而不受干擾，此功能預設為**關閉**。

---

## 啟用番茄鐘

### 桌面版
1. 開啟**設定**。
2. 前往 **GTD**。
3. 在**功能**中開啟**番茄鐘**。
4. 開啟**專注**檢視。

### 行動版
1. 開啟**選單 → 設定**。
2. 前往 **GTD**。
3. 在**功能**中開啟**番茄鐘**。
4. 開啟**專注**分頁。

---

## 運作方式

- 可選擇將目前「專注」候選項目之一設為**計時任務**，也可以保留為**只使用計時器**。
- 選擇預設組合：`15/3`、`25/5`、`50/10`，或在**設定 → GTD**中設定一組選用的自訂組合
- 開始專注時段。
- 專注時段結束時，Mindwtr 會切換至休息並暫停，讓你自行決定何時開始。
- 休息結束時，會切回專注。

連結任務是選用功能，預設為關閉。若想在「專注」中使用「計時任務」選擇器與**將任務標示為已完成**動作，請開啟**設定 → GTD → 番茄鐘 → 將計時器連結至任務**。

桌面版可收合面板：按下標題列中的箭頭，面板會縮成只顯示剩餘時間與階段的細列，計時器仍會繼續運作。系統會記住收合狀態；按下任務的播放按鈕時，面板會再次展開。

控制項：
- **計時器**：開始／暫停、重設與切換階段
- **更新任務**：將任務標示為已完成，完成所選的「計時任務」

任務選擇器控制番茄鐘面板要連結至哪項任務（若有）。計時器控制項只影響計時器。只有連結「計時任務」後才能使用**將任務標示為已完成**；此動作會將所選任務移至「已完成」，並從今日專注項目中移除。

## 已用時間

任務編輯器中，「預估時間」旁有選用的**已用時間**值。「已用時間」需要 v1.1.0 或更新版本；v1.0.5 等舊版並無此功能。

「已用時間」採用選用設計，預設為關閉。只有同時啟用**番茄鐘**功能與**將計時器連結至任務**時，相關介面才會出現：包括桌面版與行動版的編輯器欄位與任務列徽章，以及桌面版的快速開始按鈕。關閉任一功能後，任何位置都不會顯示時間相關資訊（已記錄的值會保留，重新啟用後再次顯示）。在任務編輯器版面中隱藏「預估時間」欄位，也會一併隱藏「已用時間」。

- 每當連結「計時任務」的專注時段完成，就會將該時段的專注分鐘數加到任務總計。
- 也可在編輯器中手動設定或修正總計。
- 已記錄時間的任務會在桌面版與行動版任務清單顯示小徽章，總計也會像其他任務欄位一樣在裝置間同步。

本功能刻意不提供逐次時段日誌、報表畫面或自由運行的碼錶。每項任務只有一個總計數字。如需外部報表，可透過 Cloud API 或 MCP 伺服器讀取 `timeSpentMinutes`。

## 從任務開始時段（桌面版）

同時啟用「番茄鐘」與**將計時器連結至任務**後，將游標移至任務列上會顯示播放按鈕：

- 按一下即可連結該任務並立即開始專注時段，不必再從「計時任務」下拉選單尋找。
- 按鈕也會顯示你已在該任務上完成多少個專注時段。

---

## 專注與下一步行動

**專注**檢視並不是完整「下一步行動」清單的一對一副本，而是「執行」階段的儀表板：

- **今日專注**：你明確標示為今天要做的任務，數量不超過設定的專注上限
- **今日**：今天到期、逾期或今天開始的下一步任務
- **下一步行動**：目前可執行的下一步任務
- **待回顧**：需要注意的等待中／提醒項目

「專注」刻意隱藏未來才開始的任務，以及循序專案中較後面的任務，讓檢視只呈現現在能採取行動的工作。若想查看更完整的任務清單，包括未來開始的任務，請使用**情境**、**專案**或**搜尋**。

---

## 適用時機

- 想在目前的 GTD 專注工作流程中使用輕量計時器，而不必離開頁面時使用。
- 如果偏好讓「專注」維持純清單介面，不顯示計時器，請保持關閉。
- 將它視為協助執行下一步行動的工具，而不是另一套規劃系統。

## 實用模式

- 使用 `15/3` 清理收集箱、處理小型行政工作或突破停滯。
- 使用 `25/5` 作為一般下一步行動的預設日常節奏。
- 任務已足夠明確、可以開始時，使用 `50/10` 進行專案深度工作。
- 若需要不同節奏，可使用一組自訂預設，但請保持簡單。
- 只有連結的「計時任務」確實完成時，才使用**將任務標示為已完成**；否則請暫停或切換階段，並保留任務。

---

## 注意事項

- 番茄鐘可單純作為計時器，也可連結至「計時任務」，直接從面板完成該任務。
- 面板刻意採用選用設計，偏好乾淨 GTD 專注頁面的使用者可以保持隱藏。
- 內建預設保持固定且簡單。Mindwtr 只容許加入一組選用的自訂預設，以免「專注」變成計時器自訂畫面。
