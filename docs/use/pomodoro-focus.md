# Pomodoro Focus

Mindwtr includes an optional Pomodoro panel in Focus. It is off by default so Focus stays clean for people who only want a GTD action list.

## Enable it

Turn on Pomodoro from GTD settings on desktop or mobile, then open Focus.

## How it works

- Choose a preset such as `15/3`, `25/5`, or `50/10`.
- Optionally link the timer to a Focus task.
- Start, pause, reset, or switch phase.
- Mark the linked task done only when it is actually complete.

## When to use it

Use Pomodoro when a time box helps you start or stay with a clear next action. Keep it off if timer UI makes Focus feel noisy.
