# Áreas y personas

Usa **Áreas** para las responsabilidades continuas y **Personas** para el trabajo delegado o centrado en alguien. Puedes crear ambas al asignarlas y administrarlas más adelante en Ajustes.

## Áreas

Las áreas agrupan proyectos relacionados y tareas independientes bajo una responsabilidad estable, como Trabajo, Hogar, Salud, Investigación o el nombre de un cliente.

Usa un área para responder: «¿Qué trabajo pertenece a esta parte de mi vida o mi actividad profesional?».

### Crear un área al asignarla

No tienes que salir del flujo de tareas para crear un área.

1. Abre Adición rápida, Captura rápida o el editor de tareas.
2. Abre el campo **Área**.
3. Escribe el nombre del área.
4. Si todavía no existe, elige **Crear**.
5. La nueva área queda seleccionada para la tarea.

En el escritorio, los selectores de áreas ofrecen el mismo comportamiento de creación mientras escribes. En dispositivos móviles, el selector de Captura rápida y edición de tareas te permite buscar primero y crear después el área si no hay una coincidencia exacta.

### Crear o editar áreas en Ajustes

Usa Ajustes para administrar directamente las áreas guardadas.

1. Abre **Ajustes**.
2. Abre **Administrar**.
3. Despliega **Áreas**.
4. Añade un área, cambia su nombre o color, o elimina una que ya no uses.

Al eliminar un área se conservan sus tareas y proyectos. Mindwtr los deja sin asignar en vez de eliminar el trabajo.

## Personas

Las personas son nombres de responsables reutilizables para tareas delegadas, elementos En espera y búsquedas centradas en alguien. También pueden contener notas y enlaces de referencia, como una dirección de correo, una página de contacto o un enlace de Obsidian.

Usa Personas para responder: «¿Quién es responsable, de quién estoy esperando algo o con quién debo hablar de esto?».

### Crear una persona al asignarla

1. Abre una tarea.
2. Busca **Asignada a** o **Responsable / Persona**.
3. Escribe el nombre de la persona.
4. Si aún no está guardado, elige **Nueva persona**.
5. La persona se guarda y se asigna a la tarea.

También puedes escribir un nombre sin crear la persona guardada para usarlo como responsable de una tarea. **Nueva persona** hace que el nombre se pueda reutilizar en futuras sugerencias y en Ajustes.

### Asignar una persona al capturar

En Adición rápida, escribe `%` seguido del nombre: `Ask %Jim for the budget /waiting`. Los nombres conocidos se reconocen sin comillas, incluidos los de varias palabras; escribe un nombre nuevo de varias palabras como `%"Jim Smith"`. En el escritorio, al escribir `%` se abren las mismas sugerencias que en el campo Asignada a.

### Crear o editar personas en Ajustes

1. Abre **Ajustes**.
2. Abre **Administrar**.
3. Despliega **Personas**.
4. Añade una persona, cambia su nombre, añade notas o un enlace de referencia, o elimina una que ya no uses.

Al eliminar una persona se conservan las tareas. Mindwtr borra el registro guardado de esa persona en vez de eliminar el trabajo.

## Campos relacionados

| Campo | Para qué usarlo | Dónde crearlo |
| --- | --- | --- |
| **Área** | Ámbitos de responsabilidad para proyectos y tareas | Selector de área, Ajustes → Administrar → Áreas |
| **Persona** | Responsable delegado o persona de la que esperas algo | Escribe `%person` en Adición rápida, campo Asignada a, Ajustes → Administrar → Personas |
| **Contexto** | Dónde, con qué herramienta o con quién puedes hacer la tarea | Escribe `@context` en Adición rápida o al editar la tarea |
| **Etiqueta** | Tema, energía, modo o rótulo de agrupación | Escribe `#tag` en Adición rápida o al editar la tarea |
