# Desktop keyboard shortcuts

Mindwtr desktop supports keyboard-first use, including Vim and Emacs-style presets.

## Quick start

- Choose a preset in Settings.
- Press `?` in the app to open the current shortcut sheet.
- Use `/` to search.
- In Vim mode, use `gi` for Inbox, `gn` for Next Actions, and `gf` for Focus.
- Use `j` and `k` to move selection, `e` to edit, `x` to toggle done, and `dd` to delete in Vim mode.
- Use `Ctrl+Alt+S` to run manual sync on desktop.

Shortcuts are ignored while typing in editable fields when the shortcut would conflict with text entry.

## Global quick add

Change or disable the global Quick Add shortcut in Settings. Global shortcuts depend on desktop platform permissions and packaging.
